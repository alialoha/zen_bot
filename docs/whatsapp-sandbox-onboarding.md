# WhatsApp Sandbox Onboarding

Use this template to onboard friends/testers to the Twilio WhatsApp Sandbox.

## Shareable Message Template

Copy and send this message:

1. Save this number in WhatsApp: `+1 415 523 8886`
2. Send this exact one-time join code to that number: `join <your-sandbox-code>`
3. Wait for confirmation from Twilio Sandbox.
4. Send any message after that and Zen Bot will reply.

## Notes

- Each tester must send the join code once before chatting.
- Join codes can change if Sandbox settings are reset.
- For unrestricted public access without join codes, migrate from Sandbox to a production WhatsApp sender.
