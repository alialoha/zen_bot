# Showcase Card Copy

## Title

Zen Bot - Multi-Channel AI Poet Assistant

## Subtitle

Policy-driven Telegram + WhatsApp chatbot with retrieval-grounded Persian poetry responses, productionized on Cloud Run.

## Impact Bullets

- Built a unified webhook architecture for Telegram and WhatsApp with shared routing, intent classification, and response policy controls.
- Implemented retrieval grounding over curated Rumi/Hafiz/Khayyam excerpts with citation-aware behavior to reduce hallucinations.
- Added provider abstraction and fallback strategy for low-cost LLM operations, with structured logging for traceability.
- Shipped a reproducible quality workflow (`typecheck + build + eval`) and operational runbook for deployment/debugging.
- Deployed as a public service with stable HTTPS webhooks and cloud-native runtime readiness.
